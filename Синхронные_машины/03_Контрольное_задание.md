# Контрольное расчётное задание

Курс «Синхронные машины» · выполняется после Модуля II (Лекции 1–7)

---

## Цель

Закрепить расчёт параметров и характеристик синхронного генератора: ЭДС, синхронное сопротивление, векторная диаграмма, угол нагрузки, мощность, номинальное изменение напряжения.

Вариант выбирается по номеру студента в журнале.

---

## Состав задания

Для **неявнополюсного** синхронного генератора (если не указано иное), работающего на автономную нагрузку и затем на сеть, требуется:

1. Определить число пар полюсов `p` по частоте `f` и частоте вращения `n` (`n = 60 f / p`).
2. Рассчитать фазную ЭДС холостого хода `E₀ = 4.44 f w k_об Φ`.
3. По заданным ХХХ и ХКЗ определить синхронное сопротивление `X_d` и ОКЗ.
4. Построить векторную диаграмму для номинальной нагрузки при заданном `cos φ`; найти угол нагрузки `θ`.
5. Рассчитать угловую характеристику `P(θ)`, максимальную мощность `P_max` и перегрузочную способность `k_п`.
6. Определить номинальное изменение напряжения `ΔU%` при сбросе нагрузки.
7. Сделать вывод о запасе статической устойчивости.

## Пример выполнения (образец)

**Дано:** `S = 100 кВ·А`, `U_л = 400 В`, `f = 50 Гц`, `n = 1500 об/мин`, `cos φ = 0.8` (инд.), `X_d = 1.0` о.е.

**1. Число пар полюсов:**
```
p = 60 f / n = 60·50 / 1500 = 2
```

**2. Номинальные фазные величины:**
```
U_ф = U_л/√3 = 400/1.732 = 231 В
I_ном = S/(√3·U_л) = 100000/(1.732·400) ≈ 144 А
Z_баз = U_ф/I_ном = 231/144 ≈ 1.6 Ом
X_d = X_d* · Z_баз = 1.0·1.6 = 1.6 Ом
```

**3. ЭДС при номинальной нагрузке** (`E₀ = U + jI X_d`, ток отстаёт на `φ = 36.87°`):
```
I = 144·(0.8 − j0.6) = 115.2 − j86.4  А
jI X_d = j·1.6·(115.2 − j86.4) = 138.2 + j184.3
E₀ = 231 + 138.2 + j184.3 = 369.2 + j184.3
|E₀| = √(369.2² + 184.3²) ≈ 412.6 В
```

**4. Угол нагрузки:**
```
θ = arctg(184.3 / 369.2) ≈ 26.5°
```

**5. Угловая характеристика и перегрузка:**
```
P_max = m U_ф E₀ / X_d = 3·231·412.6/1.6 ≈ 178.7 кВт
P_ном = S·cos φ = 100·0.8 = 80 кВт
k_п = P_max/P_ном ≈ 2.23   (или 1/sin θ ≈ 1/sin26.5° ≈ 2.24)
```

**6. Номинальное изменение напряжения:**
```
ΔU% = (E₀ − U_ф)/U_ф ·100% = (412.6 − 231)/231 ·100% ≈ 78.6%
```
(Большое значение типично для `X_d* = 1.0` и индуктивной нагрузки → необходим АРВ.)

**7. Вывод:** запас по углу `90° − 26.5° = 63.5°`, перегрузочная способность `≈ 2.2` — устойчивость обеспечена.

## Требования к оформлению

- Расчёт с формулами и подстановкой чисел.
- Векторная диаграмма в масштабе.
- График угловой характеристики `P(θ)` с отметкой рабочей точки и `P_max`.
- Краткие выводы.

Рекомендуемый объём — 5–8 страниц.
