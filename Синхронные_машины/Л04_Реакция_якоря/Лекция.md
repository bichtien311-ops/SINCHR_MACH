# Лекция 04. Реакция якоря и синхронное сопротивление

**Модуль II. Синхронный генератор: установившиеся режимы**

---

## 1. Что такое реакция якоря

При холостом ходе поле в машине создаёт только ротор (поток возбуждения `Φ₀`). Как только генератор нагружают, по обмотке статора течёт ток `I`, который сам создаёт магнитное поле — **поле реакции якоря** `Φ_я`.

**Реакция якоря** — это воздействие магнитного поля тока статора на основное поле машины. Результирующий поток `Φ = Φ₀ + Φ_я` (геометрически) отличается от потока холостого хода и по величине, и по положению.

Характер реакции якоря зависит от **фазового сдвига** между током `I` и ЭДС `E₀`, то есть от характера нагрузки (`cos φ`).

---

## 2. Зависимость от характера нагрузки

Рассмотрим, как ток статора влияет на основное поле при разных нагрузках:

| Нагрузка | Сдвиг `I` относительно `E₀` | Действие реакции якоря |
|----------|------------------------------|------------------------|
| Активная (`cos φ = 1`) | ток совпадает с `E₀` | **поперечная** (искажает поле) |
| Чисто индуктивная (`φ = 90°`) | ток отстаёт на 90° | **продольная размагничивающая** (ослабляет поле) |
| Чисто ёмкостная (`φ = −90°`) | ток опережает на 90° | **продольная намагничивающая** (усиливает поле) |
| Смешанная | промежуточный угол | сочетание продольной и поперечной составляющих |

**Вывод.** Индуктивная нагрузка размагничивает машину (напряжение «проседает» сильнее), ёмкостная — подмагничивает (напряжение может даже расти). Это объясняет вид внешних характеристик (Лекция 7).

---

## 3. Продольная и поперечная составляющие

Ток якоря раскладывают на две составляющие по осям ротора:

- **`I_d`** — продольная (по оси полюса `d`); создаёт продольную реакцию (намагничивающую или размагничивающую);
- **`I_q`** — поперечная (по оси `q`, между полюсами); создаёт поперечную реакцию (искажает, «скашивает» поле).

```
I_d = I · sin ψ,    I_q = I · cos ψ
```

где `ψ` — угол между током `I` и ЭДС `E₀`. Это разложение лежит в основе теории двух реакций для явнополюсной машины (Лекция 6).

---

## 4. Синхронное сопротивление (неявнополюсная машина)

В неявнополюсной машине зазор равномерный, поэтому реакцию якоря удобно учитывать как индуктивное сопротивление. Полное **синхронное сопротивление** по продольной оси:

```
X_d = X_a + X_σ
```

где:
- `X_a` — сопротивление реакции якоря (учитывает поле `Φ_я`, замыкающееся через основной магнитный путь);
- `X_σ` — индуктивное сопротивление рассеяния обмотки статора.

Активным сопротивлением обмотки `R_a` у крупных машин часто пренебрегают (`R_a ≪ X_d`).

---

## 5. Схема замещения и уравнение напряжения

Одну фазу неявнополюсного генератора представляют ЭДС `E₀` за синхронным сопротивлением `X_d`:

```
E₀ = U + j·I·X_d + I·R_a
```

При `R_a ≈ 0`:
```
E₀ = U + j·I·X_d
```

Здесь:
- `E₀` — ЭДС холостого хода (наводится потоком возбуждения);
- `U` — напряжение на зажимах фазы;
- `I·X_d` — внутреннее падение на синхронном сопротивлении.

Это уравнение — основа векторной диаграммы (Лекция 5) и расчёта характеристик.

---

## 6. Численный пример

**Задача.** Неявнополюсный генератор, фаза: `U = 6000 В`, `I = 400 А`, `cos φ = 0.8` (инд.), `X_d = 8 Ом`, `R_a ≈ 0`. Найти ЭДС `E₀`.

**Решение.** Примем `U` по действительной оси: `U = 6000`. Ток отстаёт на `φ = 36.87°`:
```
I = 400·(cos(−36.87°) + j·sin(−36.87°)) = 320 − j240  А
j·I·X_d = j·8·(320 − j240) = j·2560 + 1920 = 1920 + j2560
E₀ = 6000 + 1920 + j2560 = 7920 + j2560
|E₀| = √(7920² + 2560²) ≈ 8324 В
```

ЭДС заметно больше напряжения — индуктивная нагрузка требует повышенного возбуждения, чтобы держать `U`.

---

## 7. Выводы

1. Реакция якоря — влияние поля тока статора на основное поле машины.
2. Индуктивная нагрузка размагничивает, ёмкостная — намагничивает, активная — искажает поле.
3. Ток раскладывается на продольную `I_d` и поперечную `I_q` составляющие.
4. У неявнополюсной машины реакция якоря учитывается синхронным сопротивлением `X_d = X_a + X_σ`.
5. Уравнение: `E₀ = U + jI X_d` (при `R_a ≈ 0`).
