# Лабораторная 04. Синхронный двигатель: угловая характеристика и момент

**Тема:** Исследование синхронного двигателя: пуск, угловая характеристика момента `M(θ)`, статическая устойчивость, регулирование `cos φ`.
**Закрепляет лекции:** 09–11.

## Цель
Снять угловую характеристику `M(θ)`, определить максимальный момент и перегрузочную способность; показать асинхронный пуск с втягиванием в синхронизм; продемонстрировать регулирование `cos φ` возбуждением.

## Теория и расчёт (выполнить до симуляции)
Момент неявнополюсного двигателя:
```
M = (m·U·E₀/(Ω·X_d))·sin θ = M_max·sin θ
M_max = m·U·E₀/(Ω·X_d),   Ω = 2πf/p
k_п = M_max/M_ном = 1/sin θ_ном
```
Для явнополюсной машины добавляется реактивная составляющая `∝ sin 2θ`.

**Дано:** машина из `Лаб01` в двигательном режиме; `U_ф = 231 В`, `f = 50 Гц`, `p = 2` (`Ω = 157 рад/с`), `X_d`, `E₀` — из предыдущих работ.

Предварительно рассчитать `M_max` и `k_п` для `θ_ном = 25°`.

## Часть A. Модель (Simulink)

### A1. Схема
1. `Synchronous Machine` (двигательный режим) + `powergui` (Continuous).
2. Питание статора: `Three-Phase Source` (сеть `400 В`, `50 Гц`).
3. Возбуждение `Vf` — регулируемое.
4. Механическая нагрузка на валу: регулируемый момент `Tm` (вход момента нагрузки).
5. Измерители: `θ` (угол нагрузки / load angle), электромагнитный момент `Te`, `I_abc`, `P, Q`.

### A2. Асинхронный пуск
1. Подать на статор напряжение при невозбуждённом роторе (демпферная обмотка обеспечивает асинхронный момент).
2. Наблюдать разгон до подсинхронной скорости.
3. Подать возбуждение `Vf` → ротор втягивается в синхронизм. Зафиксировать переход.

### A3. Угловая характеристика
1. Установить номинальное возбуждение.
2. Ступенями увеличивать момент нагрузки `Tm` от 0 до выпадения из синхронизма.
3. Для каждого `Tm` записывать установившийся угол `θ` и электромагнитный момент `Te`.
