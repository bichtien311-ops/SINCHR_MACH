# Лабораторная 01. Характеристики ХХ и КЗ. Определение X_d и ОКЗ

**Тема:** Снятие характеристик холостого хода и короткого замыкания синхронного генератора, определение синхронного сопротивления `X_d` и отношения короткого замыкания (ОКЗ).
**Закрепляет лекции:** 01–05.

## Цель
Построить ХХХ `E₀(I_в)` и ХКЗ `I_к(I_в)`, по ним определить `X_d` (насыщенное и ненасыщенное) и ОКЗ; сравнить с паспортными данными модели.

## Теория и расчёт (выполнить до симуляции)
Из уравнения машины при КЗ (`U = 0`, `R_a ≈ 0`): `E₀ = I_к·X_d`, откуда
```
X_d = E₀ / I_к   (при одном и том же I_в)
ОКЗ = I_в0 / I_вк ≈ 1 / X_d*
```
Базовое сопротивление: `Z_баз = U_ном.ф / I_ном`.

**Дано (вариант по умолчанию):** трёхфазный генератор `S = 8.1 кВ·А`, `U_л = 400 В`, `f = 50 Гц`, `p = 2`. Фазное `U_ном.ф = 231 В`, `I_ном = S/(√3·U_л) ≈ 11.7 А`.

Предварительно: `Z_баз = 231 / 11.7 ≈ 19.7 Ом`. Ожидаемое `X_d* ≈ 1.0` (уточнить по эксперименту).

## Часть A. Подготовка модели (Simulink / Simscape Electrical)

### A1. Блоки
1. `Synchronous Machine Salient Pole` (или `Round Rotor`) — основная машина.
2. `powergui` (режим Continuous) — обязателен.
3. Источник механического вращения: на вход `Pm`/`w` задать постоянную скорость (синхронную) — для генератора удобнее задавать скорость через `Machine measurement`.
4. Управляемый источник тока возбуждения `Vf`/`If` — задаётся ступенями.
5. Измерители: `V_abc`, `I_abc`, блоки `RMS`.

### A2. Режим холостого хода
1. Зажимы статора **разомкнуты** (или нагружены очень большим сопротивлением).
2. Скорость ротора = синхронной (`n = 1500 об/мин` при `p=2`).
3. Меняя ток возбуждения `I_в` ступенями (0 → 1.2·I_в.ном), снимаем фазную ЭДС `E₀`.
4. Заносим точки в таблицу ХХХ.

### A3. Режим короткого замыкания
1. Зажимы статора замкнуть накоротко (через малое сопротивление/измеритель тока).
2. Скорость = синхронной.
3. Меняя `I_в`, снимаем ток статора `I_к` (осторожно, не превышать ~1.2·I_ном).
4. Заносим точки в таблицу ХКЗ.

### B3. Построение и расчёт
1. Построить обе характеристики `E₀(I_в)` и `I_к(I_в)` на одном графике (две оси).
2. Выбрать `I_в`, при котором по ХХХ получается `E₀ = U_ном.ф`; снять соответствующий `I_к`.
3. Рассчитать `X_d = E₀ / I_к`.
4. Определить **ненасыщенное** `X_d` по продолжению линейного участка ХХХ.
5. Рассчитать `X_d* = X_d / Z_баз` и `ОКЗ ≈ 1 / X_d*`.
