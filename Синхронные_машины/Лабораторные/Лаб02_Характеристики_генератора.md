# Лабораторная 02. Внешние и регулировочные характеристики генератора

**Тема:** Снятие внешних `U(I)` и регулировочных `I_в(I)` характеристик синхронного генератора при разных характерах нагрузки.
**Закрепляет лекции:** 04, 06–07.

## Цель
Экспериментально получить внешние и регулировочные характеристики при `cos φ = 1`, `cos φ = 0.8` (инд.) и `cos φ = 0.8` (емк.); объяснить их вид через реакцию якоря; рассчитать номинальное изменение напряжения.

## Теория и расчёт (выполнить до симуляции)
Уравнение неявнополюсного генератора: `E₀ = U + j·I·X_d` (при `R_a ≈ 0`).
Номинальное изменение напряжения:
```
ΔU% = (E₀ − U_ном)/U_ном · 100%
```
Ожидаемо: при индуктивной нагрузке `U` падает сильнее, при ёмкостной — может расти.

**Дано:** генератор из `Лаб01` (`U_ном.ф = 231 В`, `I_ном ≈ 11.7 А`, `X_d` — из `Лаб01`).

Предварительный расчёт `E₀` для `I = I_ном` при трёх `cos φ` (использовать `X_d` из Лаб01):
- активная: `E₀ = √((U)² + (I·X_d)²)` (приближённо);
- индуктивная/ёмкостная: учесть знак реактивной составляющей.

## Часть A. Модель (Simulink)

### A1. Схема
1. `Synchronous Machine` + `powergui` (Continuous).
2. Привод вала: постоянная скорость = синхронной.
3. Возбуждение: регулируемый источник `Vf`.
4. Нагрузка: трёхфазный `Series RLC Load` с настройкой `cos φ` (активная / RL / RC).
5. Измерители `V_RMS`, `I_RMS`, блок измерения `cos φ` (или Power measurement `P, Q`).

### A2. Внешняя характеристика
1. Установить `I_в` так, чтобы при холостом ходе `U = U_ном`.
2. **Не меняя `I_в`**, ступенями увеличивать нагрузку (ток `I` от 0 до `I_ном`).
3. Для каждого значения записать `U` и `I`. Повторить для трёх `cos φ`.

### A3. Регулировочная характеристика
1. Для каждого значения тока нагрузки `I` подбирать `I_в` так, чтобы **удержать** `U = U_ном`.
2. Записать `I_в(I)`. Повторить для трёх `cos φ`.
